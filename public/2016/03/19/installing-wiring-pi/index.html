installing wiring pi

Sat, Mar 19, 2016

I want to giving Wiring Pi a shot while experimenting sending variable amounts of power to motors that I’m driving from a Rasperry Pi.

The Python RPIO library only seems to expose HIGH and LOW (ON, or OFF). In my case I want to be able to power motors at less than 100%.

sudo apt-get install wiringpi python-dev
sudo pip install wiringpi2
